## Pipeline:

#### Collector - Tavily real-time web search + citations

#### Synthesizer - 
    Structured sections:
    Product
    Market gap
    Architecture
    Benefits
    Partner implications

#### Strategist: 
    roadmap
    competitive positioning
    enterprise insights

#### Evaluator: Refines & improves output

#### Diagram Generator: Creates visual architecture diagram


## Architecture:

User Input
    ↓
Tavily Search Agent (real-time web)
    ↓
RAG Memory (retrieve past insights)
    ↓
Synthesis Agent (structured research)
    ↓
Strategy Agent (consulting output)
    ↓
Evaluation Agent (refined output)
    ↓
Diagram Generator (visual architecture)
    ↓
Outputs:
    - UI
    - PDF
    - PPT
    - Memory storage
